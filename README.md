# nmap-scanning
Practical Nmap network scanning exercises with , screenshots, and scan results for cybersecurity learning
Objectives 
using OS(windows 7,server 2016,windows 10 ,windows 11,kali linux,vm)
- Learn the fundamentals of network reconnaissance.
- Identify live hosts on a network.
- Discover open TCP and UDP ports.
- Detect running services and versions.
- Perform operating system detection.
- Understand different Nmap scan techniques.
- Document scan results and observations
- 
- Scan approaches
- Host Discovery
- TCP SYN Scan
- Service Version Detection
- Operating System Detection
- Aggressive Scan
- UDP Scan
- Full Port Scan
- Saving Scan Results
